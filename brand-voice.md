# RiseIQ — Brand Voice Guide

> **Purpose:** This file defines how RiseIQ sounds in every piece of copy — UI microcopy, emails, error messages, onboarding, and marketing content. Read this before writing anything that a student or visitor will see.

---

## 1. Brand Identity

| Field | Value |
|---|---|
| **Product name** | RiseIQ |
| **Tagline** | Rise Smarter. Learn Faster. |
| **AI engine name** | Synapse Brain (never "AI") |
| **Target audience** | Immigrants, international students, and working professionals aiming to improve English, tech skills, or career outcomes |
| **Primary market** | Canada — English-speaking, multicultural, pragmatic |

---

## 2. Brand Personality

RiseIQ is a **coach, not a teacher.** It's direct, motivating, and results-focused — never corporate, never condescending. Think of the tone as a smart friend who has already passed the IELTS exam, gotten the job, and wants to help you do the same.

| Trait | What it means in practice |
|---|---|
| **Ambitious** | We believe students can hit Band 7.5+. Say so. Don't hedge. |
| **Warm** | Students are often stressed and far from home. Show you understand that. |
| **Direct** | No filler. No corporate speak. Every sentence earns its place. |
| **Clear** | Plain English. If a 16-year-old can't understand it, rewrite it. |
| **Motivated** | Progress is celebrated. Struggle is normalised. Giving up isn't mentioned. |

---

## 3. Voice in One Sentence

**Ambitious but human — the smart friend who's already done it.**

---

## 4. Tone by Context

| Context | Tone | Example |
|---|---|---|
| Onboarding / welcome | Warm, exciting | "You're in. Let's get started." |
| Dashboard / progress | Encouraging, specific | "3 sessions this week. Keep it up." |
| Error messages | Calm, helpful, not apologetic | "Something went wrong. Try again or contact support." |
| Practice prompts | Clear, task-focused | "Write at least 250 words. You have 40 minutes." |
| Synapse Brain feedback | Expert, constructive | "Your coherence score is 6.5. Here's exactly what to improve." |
| Empty states | Motivating, not clinical | "No sessions yet — your first one takes 10 minutes." |
| Marketing / landing page | Bold, aspirational | "Band 7.5 isn't luck. It's a system." |
| Admin portal | Professional, neutral | Short sentences. No enthusiasm. Data-first. |

## 5. Words We Use

| Use this | Not this |
|---|---|
| Synapse Brain | AI, artificial intelligence, chatbot, GPT, Claude |
| Band score | Score, grade, mark |
| Practice session | Exercise, task, activity, quiz |
| Feedback | Correction, marks, criticism |
| Learning Hub | Course catalogue, library |
| Enrolled | Registered, signed up |
| Progress | Results, performance |
| Rise | Improve, increase, boost (in headlines — "Rise" is ours) |
| Module | Chapter, unit, section |
| Mock test | Practice exam, sample test |

---

## 6. Words We Never Use

- **AI** — always "Synapse Brain" or "Synapse" in the UI
- **Algorithm** — users don't care; say what it does instead
- **Revolutionary / Cutting-edge / Next-generation** — overused, meaningless
- **Simply / Just / Easily** — condescending to people who are struggling
- **Unlock** — overused in edtech; avoid
- **World-class** — claim it with results, don't say it
- **Synergy, leverage (as a verb), empower** — corporate filler

---

## 7. Formatting Rules

- **Sentences over bullet points.** Paragraphs feel human; bullet points feel like terms of service.
- **Short sentences in UI.** Mobile-first. Every word earns its place.
- **No exclamation marks in error states.** Errors aren't exciting.
- **No ALL CAPS for emphasis** — use bold or restructure the sentence.
- **Numbers, not words, for scores.** "Band 6.5" not "Band six-point-five."
- **No Oxford comma** — "Reading, Writing and Listening" not "Reading, Writing, and Listening."
- **Product name:** always "RiseIQ" — never "Riseiq", "Rise IQ", or "riseiq."
- **Synapse Brain:** always capitalised. Never "synapse brain."

## 8. UI Microcopy Examples

### Buttons and CTAs

| ✅ Use | ❌ Avoid |
|---|---|
| Start Session | Begin / Launch / Go |
| Continue | Resume Learning |
| View Feedback | See Results |
| Browse Courses | Explore Our Catalogue |
| Get Synapse Feedback | Ask AI |
| Try Again | Retry |

### Empty States

| Page | ✅ Copy |
|---|---|
| No sessions completed | "No sessions yet. Your first one takes 10 minutes." + "Start Practicing →" |
| No courses enrolled | "You're not enrolled in any courses yet." + "Browse Courses →" |
| Synapse (coming soon) | "Synapse Brain is almost ready. You'll be the first to know." |
| Admin — no students | "No students yet. Invite someone to get started." |

### Error Messages

| Error | ✅ Copy |
|---|---|
| Wrong credentials | "That email or password doesn't match. Try again." |
| Network error | "Something went wrong. Check your connection and try again." |
| Session expired | "Your session timed out. Sign in again to continue." |
| Evaluation failed | "Synapse Brain couldn't evaluate this right now. Try again in a moment." |
| 404 | "This page doesn't exist. Head back to your dashboard." |

### Form Labels & Placeholders

| ✅ Use | ❌ Avoid |
|---|---|
| Email address | Email |
| Password | Enter your password |
| Your essay (placeholder) | Write here... / Type your answer |
| Target band (placeholder) | e.g. 7.5 | |

## 9. Synapse Brain Copy Rules

Synapse Brain is RiseIQ's smartest feature. Its copy must feel authoritative but human — like a tutor who gives real, honest feedback.

- Always attribute feedback to "Synapse Brain" not "the system" or "our AI."
- Feedback should name the specific criterion: "Your Lexical Resource score is 6.0."
- Improvements should be actionable: "Use a wider range of linking words — try 'consequently' or 'as a result' instead of repeating 'so.'"
- Strengths should be specific: "Strong use of complex sentences in paragraph 2."
- Never say "Good job!" with no context. Praise the specific thing.

## 10. Email & Notification Tone

| Email type | Tone | Subject line example |
|---|---|---|
| Welcome | Warm, direct | "You're in — here's how to start" |
| Session reminder | Friendly, brief | "Your IELTS exam is 30 days away" |
| Feedback ready | Excited but calm | "Synapse Brain reviewed your essay" |
| Certificate earned | Celebratory, specific | "You've earned your IELTS Prep certificate" |
| Inactivity nudge | Honest, no guilt | "It's been a week — 10 minutes is enough to get back on track" |

---

## 11. What RiseIQ Is Not

- Not a university — don't write like an institution
- Not a tutoring agency — don't write like a service business
- Not a gamified app — don't overdo streaks, coins, or celebration animations
- Not a generic SaaS tool — everything connects back to real-world outcomes (passing IELTS, getting hired)

---

## 12. The One Test

Before publishing any copy, ask: **Would a stressed immigrant student studying at midnight find this clear, respectful, and motivating?**

If yes, ship it. If no, rewrite it.
